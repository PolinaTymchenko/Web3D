About

For my website, I created a seasonal, interactive experience inspired by Stardew Valley. I included animated 3D models of farm life items using Blender and Three.js, giving the site charm and interactivity.
I also created themed parallax banner for homepage

3D Models

I created 3D models of a parsnip, apple, and treasure chest using Blender. Each model includes animations that reflect their in-game behavior, such as the chest opening or the parsnip bouncing slightly when selected. These animations were exported as GLB files and integrated into my website using Three.js.

Design

The models are displayed in a custom Bootstrap canvas. I used Bootstrap components like Navbar and responsive grid, while custom CSS created the soft, pixel-art inspired aesthetic. I added falling leaves and a parallax effect for visual depth. Each model is placed on its own themed page with responsive background and lighting tailored per item.

Integration

I added audio and adjusted it for the homepage. For animations i worked sith physics in Blender

Interaction

Users can interact with each 3D model using orbit controls, and toggle animations or rotation. Buttons are styled with Bootstrap and connected to Three.js actions. The parsnip, for example, can be grown with a click, while the apple rotates with another.
